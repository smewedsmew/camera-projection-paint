# Camera Projection Paint

A Blender addon for camera-based projection painting workflow with EEVEE baking support.

## Features

- Project painted images back onto objects as textures using camera-based UV projection
- EEVEE baking support for fast texture baking
- Works with multiple objects/materials/textures

## Installation

1. Download the addon as a ZIP file (or clone this repository)
2. In Blender, go to `Edit > Preferences > Add-ons`
3. Click `Install...` and select the downloaded ZIP file (or navigate to the addon folder)
4. Enable the addon by checking the box next to "Camera Projection Paint"

## Usage

1. Open the 3D Viewport sidebar (press `N`)
2. Go to the **Illustration** tab
3. Select a PSD file containing your painted layers
4. Enable objects for projection
5. Auto-map or manually assign PSD layers to texture nodes
6. Update Preview to see the projection in the viewport
7. Update Bake to prepare for baking
8. Bake with EEVEE (fast) or Cycles
9. Apply the baked result to finalize

## Requirements

- Blender 4.0.0 or higher
